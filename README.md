## How to setup :

1. Open Xammp
2. Start Apache and MySql
3. Copy this Folder 'simple_crud' to (C:/xammp/htdocs)
4. Open http://localhost/phpmyadmin/
5. Import 'simple_crud.sql'
6. access link: http://localhost/simple_crud/index.php
